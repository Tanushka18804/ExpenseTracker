Smart Expense Tracker 

A Java-based Expense Tracking System that allows users to add, view, filter, and analyze their daily expenses with MySQL database support.
This project demonstrates strong concepts in Core Java, JDBC, MySQL, OOP, modular architecture, input handling, and clean code structure.
